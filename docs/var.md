# 变量和列表
***
> 变量和列表是一个语言中最基础的东西，所以在这里首先介绍变量和列表。

## 变量相关

- `var <variable>;` 创建名为`variable`的变量
- `var <variable> = <value>;` 创建变量并赋值
- `<variable> = <value>;` 变量赋值

在这里，我不得不提出一个概念，那就是本语言里，变量是不需要详细区分它的类型的变量，因为解释器会自动处理它们。所以很多时候，只要不涉及特殊情况，你都可以不用管它会发生什么类型错误。

比如你可以输入`var a = 123;`或者`var a = 3.14;`，它们都是数字；而`var a = true;`就是布尔值了。如果你选择`var a = "abc";`便是字符串——是的，字符串必须被**双引号**包起来！

字符串中会有某些特殊的转义字符，比如"\n"、"\r"、"\t"、"\d"等，基本都是支持的。

某些特殊的值，比如`NULL`(空值)、`INF`(无穷大)以及`NaN`(不是数)直接打出来就是了，就像`var a = NULL;`一样。

可用的方法:

- `.len()` 返回变量字符数
- `<variable>[n]` 返回变量第n个字符

## 列表相关

列表也是变量的一种类型：

- `var <list> = [""];` 创建名为`list`的列表并赋值
- `<list> = [""];` 使用 Array 为列表赋值
- `<list>[n] = <value>;` 将列表第n项设为`<value>`
- `<list1> + <list2>` 连接两个列表
- `<list>.push(<value>);` 将`<value>`加入列表
- `<list>.remove(n);` 删除列表第n项
- `<list>.insert(n,<value>);` 在列表第n项之前插入`<value>`
- `<list>.clear();` 清空列表

列表的每一项和变量一样。

可用的方法:

- `.len()` 返回列表长度
- `.sum()` 求列表所有项目的和
- `.in(<value>)` 判断列表是否含有某个值
- `.lin(<value>)` 返回以特定字符连接各个列表项目后的值

比如我创建了变量`var awa = 114;`，那么我就可以使用`awa`来引用它，并且`awa.len()`将返回`3`，这个方法将把`awa`的值视作为字符串返回其长度。

再比如我创建了列表`var a = ["114","514","191"];`，那么我就可以使用`a[]`来引用它，并且`a.len()`将返回`3`，这个方法将返回列表的长度；或者`a.in("114")`将返回`true`，因为列表中包含字符串`114`。需要提及的是，**列表是以`0`作为开头的**，负数会倒着返回项(比如`a[-1]`将返回`191`)。

## 常量

- `const <name> = <value>;` 创建一个常量并赋值为`<value>`

> 常量无论如何都<font color='red'>不能</font>在程序执行过程中重新赋值!!!

如果你定义了一个用户类，并且需要定义一个这个类的类型的常量实例，你可以使用`const <class> <name> = <class>();`来定义，这样你会获得一个用户类的初始化对象，因为在后面的调用的函数是类的构造函数。

## 运算和逻辑语句

- `+` `-` `*` `/` 四则运算
- `%` 求模
- `**` 求指数
- `++` `--` 加/减一
- `>` `<` 大于/小于
- `==` 等于
- `>=` `<=` 大/小于等于
- `&&` `||` `!` 与或非
- `int()` `float()` `bool()` 强制类型转换
- `type()` 判断一个值的类型(返回`int`、`float`、`bool`和`string`的字符串)

> 右值类型判断优先度: `变量/列表/常量` > `无定义常量(即直接写出来的内容)` > `结构体内容` > `运算值`

> 表达式优先级：
>    最高: ( ) [ ] . function-call
>        ! unary - ++ --
>        * / % **
>        + -
>        > < >= <=
>        == !=
>        &&
>        ||
>        ?:
>    最低: =

我是不是提到了字符串操作？哦，很显然，你可以通过`+`来连接两个字符串。比如`"Hello " + "World!"`就会把两个词连在一起。

什么？你问我如果你使用`123 + "456"`会变成什么？当然是字符串了！字符串的优先级是最高的，它们的顺序如下： `字符串` > `布尔` > `浮点型` > `整型` 

## 结构体相关

> 结构体就像一个可以存储不同类型对象和值的大型"仓库"

本语言没有结构体，但是我们有类似的东西——对象字面量：

- `var <name> = {<object>:<value>}` 定义对象字面量
- `<name>.<object>` 访问成员

在这里，`<object>`必须是一个标识符，不能是字符串等。比如：

```cpp
var awa = { x : 10,
            y : 20 };
```

值得注意的是，在表达式上下文的位置使用花括号，那么解释器将视为这是对象字面量；如果在定义类的时候使用花括号，解释器将其视为一个类的主体；语句位置的花括号永远指的是代码块。
