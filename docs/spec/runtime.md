# 运行时和值

***

> 本文定义 RadiumScript 在运行时能处理的值，以及常见运算的转换规则。解释器和编译到 JavaScript 的实现都应保持一致。

## 值类型

| 类型 | 示例 | `type()` 返回 |
| --- | --- | --- |
| 整型 | `114` | `int` |
| 浮点型 | `3.14` | `float` |
| 布尔值 | `true`、`false` | `bool` |
| 字符串 | `"abc"` | `string` |
| 列表 | `[1,2,3]` | `list` |
| 对象字面量 | `{x:1,y:2}` | `object` |
| 类实例 | `timer()` | 类名 |
| 函数 | `def test() {}` | `function` |
| 空值 | `NULL` | `null` |
| 无穷大 | `INF` | `float` |
| 非数值 | `NaN` | `float` |

`type()` 对类实例返回类名，这样脚本可以区分不同用户类实例。

## 真值规则

条件判断、`!`、`&&`、`||` 使用真值规则。

| 值 | 真值 |
| --- | --- |
| `false` | 假 |
| `NULL` | 假 |
| `0`、`0.0` | 假 |
| `NaN` | 假 |
| `""` | 假 |
| 空列表 `[]` | 真 |
| 空对象 `{}` | 真 |
| 其他值 | 真 |

列表和对象即使为空也为真，这是为了避免和 `NULL` 混淆。

## 强制转换

| 表达式 | 结果 |
| --- | --- |
| `int(3.14)` | `3` |
| `int("123")` | `123` |
| `int(true)` | `1` |
| `int(false)` | `0` |
| `float("3.14")` | `3.14` |
| `bool(0)` | `false` |
| `bool("")` | `false` |
| `bool(NULL)` | `false` |
| `string(123)` | `"123"` |

无法转换成数字的字符串转换为 `NaN`。

## 运算规则

`+` 在任意一侧为字符串时执行字符串连接。

| 表达式 | 结果 |
| --- | --- |
| `1 + 2` | `3` |
| `1 + 2.5` | `3.5` |
| `1 + "2"` | `"12"` |
| `true + 1` | `2` |
| `false + 1` | `1` |
| `NULL + 1` | `1` |
| `[1] + [2]` | `[1,2]` |

`-`、`*`、`/`、`%`、`**` 会把操作数转换为数字。无法转换时结果为 `NaN`。

| 表达式 | 结果 |
| --- | --- |
| `"5" - 1` | `4` |
| `"a" * 3` | `NaN` |
| `true * 3` | `3` |
| `NULL * 3` | `0` |
| `1 / 0` | `INF` |
| `0 / 0` | `NaN` |

比较运算中，如果两边都是数字或可转换为数字，则按数字比较；否则按字符串比较。`==` 和 `!=` 使用严格值比较，不做跨类型宽松相等。

| 表达式 | 结果 |
| --- | --- |
| `1 == 1` | `true` |
| `1 == "1"` | `false` |
| `NULL == NULL` | `true` |
| `NaN == NaN` | `false` |
| `[1] == [1]` | `false`，除非是同一个列表对象 |

## 列表和成员访问

列表索引从 `0` 开始。负数索引从末尾倒数，`-1` 表示最后一项。越界访问返回 `NULL`。

对象字面量和类实例使用 `.` 访问成员。访问不存在的成员返回 `NULL`，给不存在的对象字面量成员赋值会创建成员；给类实例不存在的成员赋值应报错。

## 常量

`const` 创建的值不能被重新赋值。若常量保存列表、对象或类实例，常量名称不能重新绑定；是否允许修改其内部成员由实现决定。为了让解释器和 JS 编译结果一致，建议常量的内部成员也不可修改。
